# vouchers-nbt.json

## Format

{% code title="vouchers-nbt.json" %}
```json
{
    "voucher-id": {
        // JSON here
    }
}
```
{% endcode %}

## Examples

A colored leather item

```json
{
    "voucher-id": {
        "display": {
            "color": 14574937
        }
    }
}
```
